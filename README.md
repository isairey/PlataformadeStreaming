# 🌌 Acerca del proyecto

**Nyetflix** es una plataforma de streaming multimedia inspirada en la experiencia visual y funcional de Netflix.

El proyecto fue creado como:

- 🎓 Proyecto escolar
- 🧪 Plataforma de práctica
- 💻 Entorno de aprendizaje Full Stack
- 🚀 Experimento de streaming moderno

Actualmente el proyecto se encuentra en una fase **Work In Progress (WIP)**, donde gran parte del frontend consiste en mockups estáticos mientras el backend continúa evolucionando.
